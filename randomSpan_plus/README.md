#RandmonSpan
###随机字体颜色，位置偏移插件

![alt text][./img.png]

参数：
*minFontSize:最小字体
*maxFontSize:最大字体
*colorArr:颜色数组
*marginLR:左右margin
*postionBT:上下postion位移
*postionLR:左右postion位移
*Unit:单位，margin与字体均使用它，默认px,可选em，rem
*newlineNum:换行条件


使用:
`
<script src="./randomSpan_plus.js"></script>
$(".test").randomSpan(options);
`
